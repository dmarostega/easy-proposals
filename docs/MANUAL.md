# Manual do Usuário — Proposta Fácil (Easy Proposal)

## Visão geral

O Proposta Fácil é um sistema para criação e acompanhamento de propostas comerciais.

## Acesso ao sistema

O usuário deve criar uma conta ou acessar com login e senha.

## Dashboard

O dashboard apresenta um resumo das propostas criadas, aprovadas e pendentes.

## Clientes

Nesta área será possível cadastrar e gerenciar clientes.

## Propostas

Nesta área será possível criar propostas, adicionar itens, definir validade e gerar link público.

## Planos

O sistema possui plano gratuito e planos pagos configuráveis.

## Observações

Este manual será atualizado conforme o desenvolvimento do sistema evoluir.
