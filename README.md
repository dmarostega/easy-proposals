# Proposta Fácio (Easy Proposal)

SaaS para criação e gestão de propostas comerciais e orçamentos para freelancers, prestadores de serviço e pequenos negócios.

## Stack

- Laravel 13
- Vue.js 3
- TypeScript
- MySQL
- Tailwind CSS
- Vite
- Inertia.js

## Objetivo

Permitir que usuários criem propostas comerciais, enviem links públicos para clientes e acompanhem aprovações.

## Funcionalidades previstas

- Autenticação
- Dashboard
- Clientes
- Serviços
- Propostas
- Link público de aprovação
- PDF
- Painel admin
- Planos configuráveis
- SEO
- robots.txt
- sitemap.xml
